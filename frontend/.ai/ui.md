# GhostOS UI Guide

## Theme
- Dark premium fitness aesthetic with Strava-like energy.
- Primary accent is GhostOS orange `#FC4C02`.
- Background layers: `#0F0F10`, `#181818`, `#1F1F21`.
- Dividers use `#2A2A2A`.

## Spacing
- Base page padding: 18 to 20 px.
- Card padding: 20 to 32 px depending on emphasis.
- Vertical rhythm: usually 12, 14, 16, 18, or 22 px gaps.

## Components
- `GhostAppBar`: profile, app title, plan badge, search, notifications.
- `GlassCard`: blurred premium surface with gradient fill and border.
- `SectionHeader`: title, subtitle, optional action label.

## Current Screens
- Home: ghost hero card, weather module, stat chips, mission carousel, quick actions.
- Activity: metric rings, overview panel, timeline items.
- Record: category pills, intensity and duration rails, primary record CTA.
- You: profile hero, compact stat cards, settings rows.
- Auth: premium login card with segmented sign-in/sign-up flow.
- Splash: centered brand card with glow orbs and progress bar.

## Notes for Future Work
- Preserve UI-only scope unless explicitly asked otherwise.
- Keep additions consistent with the existing orange-on-black visual language.
- Extend reusable widgets before adding one-off screen styling.
